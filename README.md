# AIML Internship Projects – Elevate Labs  

This repository contains all the Artificial Intelligence & Machine Learning projects completed during my **AIML Internship at Elevate Labs**.

The internship focused on building real-world machine learning solutions, improving problem-solving skills, and gaining hands-on experience in model development, evaluation, and deployment.

---

## 📌 Internship Details  

- **Domain:** Artificial Intelligence & Machine Learning  
- **Organization:** Elevate Labs  
- **Technologies Used:** Python, Machine Learning, Deep Learning, NLP, Streamlit, Flask  

---

## 🚀 Projects Included  

### 1️⃣ Movie Recommendation System  
A machine learning recommendation engine that suggests movies using collaborative and content-based filtering techniques.

**Key Features:**  
- Uses MovieLens dataset  
- Cosine similarity based recommendation  
- Streamlit web interface  
- Real-time movie suggestions  

**Tech Stack:** Python, Pandas, Scikit-learn, Streamlit  

---

### 2️⃣ Fake News Detection  
An NLP classification model that predicts whether a news article is fake or real.

**Key Features:**  
- Text preprocessing using NLP  
- TF-IDF vectorization  
- Logistic Regression / Naive Bayes  
- Model performance evaluation  

**Tech Stack:** Python, NLTK, Scikit-learn, Pandas  

---

### 3️⃣ Stock Price Prediction using LSTM  
A deep learning time-series forecasting project that predicts future stock prices using historical trends.

**Key Features:**  
- Data normalization & sequence preparation  
- LSTM neural network model  
- Prediction visualization  
- Trend analysis  

**Tech Stack:** Python, TensorFlow/Keras, Pandas, Matplotlib  

---

### 4️⃣ AI Resume Ranker  
An intelligent NLP based system that ranks resumes according to job description relevance.

**Key Features:**  
- Resume text extraction  
- TF-IDF similarity scoring  
- Candidate ranking algorithm  
- Flask based web interface  

**Tech Stack:** Python, NLP, Scikit-learn, Flask
